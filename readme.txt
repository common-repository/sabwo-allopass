== Description ==

Systeme d'abonnement des blogs wordpress (Allopass)


== Installation ==

1. Vous devez cr&eacute;e un compte marchand chez Allopass http://www.allopass.com/ puis cr&eacute;e un abonnement dans la section produits.
2. configuration compte allopass:


URL d'acha : http://www.YourURL.com/Site/  
URL d'acc&eacute;s au produit : http://www.YourURL.com/Site/wp-content/plugins/sabwo-allopass/login.php


3. D&eacute;finissez vos prix et paliers et dur&eacute;es de votre abonnement.

4. Une fois votre produit (abonnement) est configur&eacute;, reportez les informations de votre produit dans le formulaire de configuration du plugin SABWO
ID site, Id document (full numbers): 123456/123456/123456, Auth (only six number of the middle): xxxxxx/123456/xxxxxx  .

5. Ajouter WIDGET SABWO la ou vous souhaitez voir le formulaire de connexion.

6. Help file at: http://plugins.svn.wordpress.org/sabwo-allopass/trunk/doc/SABWO_help.doc  .
